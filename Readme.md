# 🎬 GitStory AI

Turn any GitHub repository into a cinematic story video.

GitStory AI transforms raw commit history into a visual, story-driven timeline — showing how a project evolved over time with commits, file changes, and milestones.

---

## 🚀 Features

- 📜 Converts commits into story-style narration
- 📦 Shows files added, modified, and removed
- 📘 Detects README/documentation evolution
- 📊 Tracks project growth (lines of code)
- 🎬 Generates a complete MP4 video timeline
- ⚡ Works directly with GitHub repo URLs
- 🏷️ Permanent watermark branding

---

## 🎥 What It Shows

- Project journey from first commit → latest
- Feature additions and bug fixes
- File-level changes
- Major breakthroughs in development
- Documentation evolution
